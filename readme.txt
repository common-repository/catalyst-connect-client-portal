== Description ==

This plugin is no longer being supported. Please visit https://zportals.com for the premium version.

The plugin utilizes data directly from the Zoho CRM and allows the user to pick and choose which data is visible on your website, customizing the appearance to look exactly how you want it. It also features different view and edit permissions along with various data types allowing your customers to be better informed and up to date with current information. Communication is key for any organization, and now you can do it with the most efficiency and professionalism using the Catalyst Connect Zoho Client Portal for WordPress. Ideal for those who want to display data from within their CRM to users in WordPress.

== Installation ==

1. Upload the plugin files to the `/wp-content/plugins/plugin-name` directory, or install the plugin through the WordPress plugins screen directly.
2. Activate the plugin through the 'Plugins' screen in WordPress.
3. Go to the new menu item labeled "Client Portal".
4. Follow the steps outlined on that page in order to connect the plugin with your Zoho CRM account.


== Frequently Asked Questions ==

= What version of WordPress will the plugin work with? =

Version 4.9.10 and above.

= Which version of PHP will the WordPress plugin work with? =

The plugin requires PHP 5.6 or newer.

= Is there a limit on the number of users who can access the portal? =

Yes, the free version of the plugin is limited to 5 users. For unlimited users, please have a look at our paid plugin: https://catalystconnect.com/zoho-client-portal-for-wordpress/

= Do I need a Zoho account in order to use the plugin? =

Yes, a Zoho One or Zoho Enterprise license is required in order to use the plugin. Zoho trial accounts will also work.

= Does the plugin integrate with other Zoho apps besides the CRM? =

The free version of the plugin only integrates with Zoho CRM. Our paid plugin integrates with many other Zoho products. You can find more details at: https://catalystconnect.com/zoho-client-portal-for-wordpress/
